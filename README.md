Step-by-Step Process:
1️⃣ Importing Data
Download and load the SuperStore dataset into Power BI
Understand data tables and their structure
2️⃣ Data Cleaning & Transformation (Power Query Editor)
Remove null values and duplicates
Change data types (date, number, text)
Create calculated columns for better insights
3️⃣ Data Modeling
Create relationships between dimension and fact tables
Apply star schema for efficient reporting
4️⃣ DAX Measures Creation
Build KPIs like Total Sales, Profit, Returned Orders, and YoY Growth
Create advanced DAX formulas to compare Previous Year vs Current Year (PY vs CY)
5️⃣ Dashboard Design & Visualization
Use charts such as Line Chart, Bar Chart, Donut Chart, and KPI Cards
Create visuals for: ✔️ Profit by Product
✔️ Sales vs Sales PY
✔️ Sales by Segment
✔️ Profit by State
6️⃣ Adding Filters and Interactions
Use slicers for easy filtering
Add user-friendly navigation and formatting
7️⃣ Publishing the Dashboard
Publish to Power BI Service
Share dashboard insights with stakeholders
✨ Highlights of the Dashboard
✔️ Total Sales, Total Profit, % Returned Orders
✔️ YoY and MoM trend comparisons
✔️ Category and segment-wise performance
✔️ State-wise profit distribution
✔️ Attractive UI/UX with color consistency
This project demonstrates how data can be transformed into meaningful insights, helping businesses make data-driven decisions.
